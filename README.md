# Task Manager
Este es un proyecto de administración de tareas desarrollado con Flask. Permite a los usuarios crear, completar y comentar tareas.

## Requerimientos

- Python 3.x
- Flask
- Flask-SQLAlchemy
- Flask-Login
- Flask-Migrate
- SQLite (base de datos por defecto)
